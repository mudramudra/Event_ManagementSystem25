Event Management System
📌 Project Overview

The Event Management System is a web-based application developed to simplify the process of organizing and managing events. 
It allows users to explore available events, register for them, and enables administrators to manage event details through 
an easy-to-use interface.

✨ Features
User Registration and Login
Browse Available Events
Event Registration
View Event Details
Admin Panel
Add New Events
Edit Event Information
Delete Events
Responsive User Interface
Secure Database Storage
